# ng-image-zoom

It's a angular wrapper for image-zoom utility. 

# Dependencies

Jquery

image-zoom.js

# Usage

```ruby
<app-image-zoom zoom-type="adjacent" image-zoom="./assets/images/1_zoom.jpg" 
image-standard = "./assets/images/1_standard.jpg" 
[image-thumbnails] = "thumbnails"
></app-image-zoom>
```
